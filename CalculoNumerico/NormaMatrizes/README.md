# 1. Norma de Frobenius: 
*(norms.py)*
 - A raiz quadrada da soma de todos os quadrados. 
 - Isso é semelhante ao comprimento “pitagórico” comum, onde o tamanho de um vetor é encontrado tirando-se a raiz quadrada da soma dos quadrados de todos os elementos.
 - Também conhecida por norma euclidiana.

# 2. Norma Infinita: 
*(norms.py)*
 - A soma máxima absoluta das linhas. 
 - Simplificando, somamos os valores absolutos ao longo de cada linha e obtemos a maior resposta.

# 3. Norma 1:
*(norms.py)*
 - A soma máxima absoluta das colunas. 
 - Simplificando, soma-se os valores absolutos em cada coluna e obtém o maior resultado.
